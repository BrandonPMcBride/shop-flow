# Shop Flow

A full-stack e-commerce application built with the PERN stack (PostgreSQL, Express, React, Node.js).

## Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS
- **Backend**: Node.js, Express (to be implemented)
- **Database**: PostgreSQL (to be implemented)

## Project Structure

```
shop-flow/
├── frontend/         # React frontend application
│   ├── src/         # React source code
│   ├── public/      # Static assets
│   └── package.json # Frontend dependencies
├── backend/         # Express backend application (to be implemented)
│   └── (Express server files will go here)
└── README.md
```

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- PostgreSQL (for backend - to be set up)

### Frontend Setup

1. Navigate to the frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Backend Setup

The backend will be set up in the `backend/` directory. (To be implemented)

## Development

This is a monorepo structure where:
- `frontend/` contains the React application
- `backend/` will contain the Express server and API
